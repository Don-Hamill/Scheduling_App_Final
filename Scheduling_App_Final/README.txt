Scheduling App Java version 1.0

The purpose of this application is to give users the ability to edit
customer and appointment information in a MySQL database via a javafx GUI.

Author: Don Hamill
Contact: dham156@wgu.edu
Version: 1.0
Date: 11/28/2021
IDE: Intellij 11.0.10
JDK: Java SE 11.0.11
JavaFX: JavaFX-SDK-11
JDBC: mysql-connector-java-8.0


- How to run the program -

Open the project in Intellij and update the paths for JavaFX and the MySQL
connector. Run the project from the main class.


- Additional report for part A3f -

When viewing appointments, the user can see the appointment schedule for a
customer; similar to contacts. Customers and contacts are selected using a combobox.
